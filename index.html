Наши достижения

Специалисты самых разных направлений в области IT обеспечивают работоспособность всей инфраструктуры предприятия

Программно-аппаратный комплекс Reeltrack

Для контроля параметров по месту проведения работ по ГРП разработан пограммно-аппаратный комплекс Reeltrack, который принимает данные напрямую с полевых контроллеров.

Программное обеспечение NetMonitoring

Программное обеспечение NetMonitoring собирает и хранит важные данные полевых работ, позволяет конечному пользователю просматривать безопасные данные в режиме реального времени с ПК, контролировать несколько заданий из одного места, получать удаленную диагностику и поддержку.

Программа Онлайн ГРП

Своевременная и полная информация с датчиков при производстве гидроразрыва пласта — залог успешного и безопасного проведения работ.

ПО «Онлайн ГРП» позволяет просматривать графики на любых устройствах, подключенных к сети интернет, выбирать интересующие параметры, перемещаться по временной шкале, сохранять данные для дальнейшего анализа.

Контроллер Онлайн ГРП

Гидроразрыв пласта — сложная и ответственная операция. Система сбора информации Онлайн ГРП реализована на базе контроллеров Siemens, отличающихся высокой надежностью и долговечностью.

Для адаптирования контроллера к работе в составе системы, в нашей компании было разработано специальное программное обеспечение в среде TIA Portal.

Контроллер для сбора
и обработки данных
с насосов высокого давления

Комплекс для сбора и обработки данных с насосов высокого давления, находящихся на объектах добычи нефти, реализован на базе контроллеров Siemens. Разработка ПО для этого контроллера велась в среде разработки TIA Portal.

Программатор датчиков высокого давления

Датчики высокого давления Viatron применяются при производстве работ по гидроразрыву пласта.
